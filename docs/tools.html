Tools & Guides

Everything you need to know about the tools included in Vibecode Toolkit

Claude Code

AI coding assistant in your terminal

Why this one

Claude Code is built specifically for AI-assisted development in the terminal. It understands your entire project context and can work with your files directly, making vibecoding locally feel as natural as prompting in a browser.

What it does

Claude Code is Anthropic's official CLI tool that brings Claude AI directly into your terminal. It helps you write code, debug issues, refactor projects, and learn new concepts without leaving your command line.

Quick start

claude

Simply type claude or use the alias c to start a conversation.

Common uses

  • Generate boilerplate code and components
  • Debug errors with context from your codebase
  • Explain complex code patterns
  • Refactor and optimize existing code

Ghostty

Fast, GPU-accelerated terminal

Why this one

Ghostty is fast, beautiful, and works perfectly right out of the box. No configuration needed to get started, but highly customizable when you're ready. It's also completely free and built for modern macOS.

What it does

Ghostty is a modern terminal emulator built for speed and aesthetics. It uses GPU acceleration for smooth rendering and comes pre-configured with sensible defaults.

Why it's great

  • Lightning-fast performance with GPU rendering
  • Beautiful out-of-the-box design
  • Native macOS integration
  • Highly customizable via ~/.config/ghostty/config

Configuration

Config file location: ~/.config/ghostty/config

Starship

Minimal, fast shell prompt

Why this one

Starship shows you exactly what you need to know (Git status, which directory you're in) without being cluttered or overwhelming. It's minimal, fast, and makes the terminal feel more helpful instead of cryptic.

What it does

Starship is a cross-shell prompt that shows you useful context about your current directory, like Git status, Node version, and command duration.

What you'll see

  • Current directory (truncated to last 3 levels)
  • Git branch and status (modified files, commits ahead/behind)
  • Command execution time (for commands over 2 seconds)
  • Color-coded success/error indicators

Configuration

Config file location: ~/.config/starship.toml

CLI Power Tools

Modern Unix tool replacements

Why these tools

These are modern, faster, and prettier versions of standard Unix commands. They do what the originals do, just better — with colors, Git integration, and smarter defaults. They make the terminal feel less intimidating and more useful.

ripgrep - Fast code search

Recursively search directories for regex patterns. Automatically respects .gitignore.

rg "function" --type js

bat - Better cat

View files with syntax highlighting and Git integration. Aliased as cat.

bat package.json

eza - Modern ls

List files with colors, icons, and Git status. Use the alias ll.

ll

fzf - Fuzzy finder

Interactive filtering for commands, files, and history.

git log | fzf

jq - JSON processor

Parse, filter, and format JSON from the command line.

curl api.example.com | jq '.data'

Shell Aliases

Quick shortcuts for common commands

Git shortcuts

  • g → git
  • gs → git status
  • gp → git push
  • gl → git pull
  • gco → git checkout
  • gcm → git commit -m
  • gaa → git add -A

Other shortcuts

  • c → claude
  • ll → eza -la --git
  • cat → bat --paging=never
  • .. → cd ..
  • ... → cd ../..
